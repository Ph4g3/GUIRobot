Class MouseInput

source code

Creates a mouse input_I structure. dx and dy are mouse coordinates. They can be left at 0, as can mouseData and time. dwExtraInfo is a pointer to a ulong data type holding extra information on our action. dwFlags is the click value of the virtual mouse. Info can be found at http://msdn.microsoft.com/en-us/library/ms927178.aspx
